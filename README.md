# 头瞄模拟器 / OpenTrack UDP数据发送器 - HeadFaker

这是一个Python + 静态网页项目，通过网页设置X、Y、Z、偏航、俯仰、滚转数值，然后由Python在特定端口用UDP广播，发送给OpenTrack，达成模拟头瞄的效果

对于一些不想买/不想设置头瞄，但是需要头瞄指向的场景（比如在星际公民里进行拍摄），或者是进行快速调试的时候可以用上

## 源代码部署

Clone到本地

```bash
# 安装Python依赖
pip install flask flask-socketio pyyaml
```

在项目目录执行
```
python server.py
```

## 配置说明

编辑 `config.yaml` 文件可以修改以下配置：

```yaml
web:
  port: 24242           # 网页服务器端口
  host: "0.0.0.0"       # 服务器地址，全0对外开放
  debug: false          # 调试模式

udp:
  port: 4242               # UDP广播端口
  host: "127.0.0.1"        # UDP目标地址
  broadcast_frequency: 20  # 广播频率 (Hz)
```

其余属性不推荐修改

### 交互方式

1. **鼠标滚轮**: 将鼠标悬停在滑动条上，滚动滚轮可调整数值
2. **拖动控制**: 点击并拖动滑块进行相对调整（不是跳转到点击位置）
3. **缩放设置**: 每个滑动条下方的缩放值控制调整灵敏度（默认1.0）
4. **循环轴**: 偏航、滚转轴为循环轴，超过180°会从-180°继续
5. **实时发送**: 每次数值变动都会自动发送到OpenTrack服务器

## 与OpenTrack配合

确保OpenTrack已正确配置：
1. OpenTrack的输入设置为`UDP over network`
2. 端口设置为 `4242`（默认就是）

## 注意事项

1. 循环轴（偏航、滚转）超过范围时会自动循环
2. 缩放值可以设置为0或任意非负数，设置为0时滑块将无法调整
